# Switch LAN Play (Linux 32-bit)

Este repositorio contiene un binario compilado del proyecto [switch-lan-play](https://github.com/spacemeowx2/switch-lan-play), optimizado y compilado para sistemas Linux de arquitectura 32 bits.

## Detalles de la compilación
* **Proyecto original:** [spacemeowx2/switch-lan-play](https://github.com/spacemeowx2/switch-lan-play)
* **Arquitectura:** Linux 32-bit
* **Propósito:** Almacenamiento y backup personal de binarios funcionales.

## Créditos
El código fuente pertenece a [spacemeowx2](https://github.com/spacemeowx2). Este repositorio solo actúa como un contenedor para el binario compilado.
